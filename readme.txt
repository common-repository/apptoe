Highly customize AppToe plugin is for Repair Stores  Wordpress Website to take appointments for their On Site, In Store and Mail in Repair orders.

== Description ==
AppToe Plugin helps you to create a page which takes appointments for your Repair Website. It adds button to the Menu of the Main Page which has an iframe code of your AppToe Store. Create your AppToe Store and get the Secret Key to add this Plugin to your Website.

== Installation ==
1. Install the AppToe Plugin.
2. Create your store at AppToe - www.goapptoe.com
3. Set the configurations and get the Secret Key from Scheduling Link.
4. Enter Name and Secret Key from your AppToe Store.
5. Set the plugin onto your website.
